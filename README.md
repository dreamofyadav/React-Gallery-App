React Tabs & Gallery App:

A modern, responsive React application featuring:

Animated tabs using Framer Motion.

Gallery with local image upload support.

Pagination for gallery images.

Smooth hover and transition effects.

Features: 

 Tabs Widget: 

Three tabs: About Me, Experiences, Recommended.

Smooth animated transitions between tabs.

Active tab highlighted with a bottom border.

Gallery Widget: 

Upload images from your local device.

Paginated display with 3 images per page.

Navigation buttons for next/previous pages.

Hover effect with subtle elevation and shadow.

Tech Stack: 

React – Frontend framework.

Framer Motion – Animations and transitions.

Tailwind CSS – Styling and responsive layouts.
